에이온 3.0 미니

Aion-3.0-Mini (AionLabs)

Aion 3.0의 경량 버전. 협업 생성 구조와 롤플레이 특화 능력은 유지하면서 비용을 풀 모델 대비 약 1/4 수준으로 낮췄다.

어떤 모델인가

Aion 3.0 Mini는 AionLabs의 다중 모델 협업 생성 구조를 그대로 유지하면서 크기를 줄인 경량 버전이다. GLM 패밀리 기반의 롤플레이·스토리텔링 특화 설계를 계승하며, 추론(reasoning) 기능도 지원한다.

컨텍스트 윈도우는 131,072 토큰, 최대 출력은 32,768 토큰으로 풀 모델과 동일하다. 가격은 입력 $0.7/1M 토큰, 출력 $1.4/1M 토큰으로, 풀 모델(입력 $3, 출력 $6) 대비 약 1/4 수준이다. 캐시 읽기는 $0.18/1M 토큰.

사용 예시

컨텍스트 길이와 출력 한도가 풀 모델과 같아서, 긴 대화형 롤플레이나 장편 스토리 작성에서도 동일하게 활용할 수 있다. 빠른 프로토타이핑이나 비용이 민감한 크리에이티브 앱에서 먼저 Mini로 개발하고, 필요하면 풀 모델로 전환하는 방식이 실용적이다. OpenAI 호환 API로 제공되어 기존 워크플로우에 바로 붙일 수 있다.

주의할 점

경량화 모델인 만큼 복잡한 다중 등장인물 서사나 고도로 세밀한 감정 묘사에서는 풀 모델 대비 출력 품질 차이가 있을 수 있다. 실제 품질은 용례에 따라 직접 비교해 보는 것이 좋다.
