Nano Banana 2 (Gemini 3.1 Flash Image)

Google DeepMind가 출시한 AI 이미지 생성 모델로, Nano Banana Pro 수준의 품질을 Gemini Flash의 속도로 제공한다.

개념 설명

Nano Banana 2는 Google DeepMind가 출시한 AI 이미지 생성 모델이다. 공식 모델명은 gemini-3.1-flash-image로, 기존 Nano Banana Pro의 고품질 생성 능력과 Gemini Flash의 빠른 속도를 한 모델에 담았다. 프롬프트 1개당 3~4초 안에 이미지를 생성하며, 이전 세대인 Nano Banana보다 텍스트 렌더링이 정확하고 같은 캐릭터나 피사체를 여러 이미지에 걸쳐 일관되게 유지하는 피사체 일관성(subject consistency)이 크게 개선됐다.

이 모델의 가장 큰 차별점은 Gemini 모델의 실시간 웹 검색과 세계 지식을 이미지 생성에 직접 활용한다는 점이다. 특정 실존 장소나 대상을 정확히 묘사해야 할 때, 학습 데이터에만 의존하지 않고 실시간 정보를 끌어와 더 정확한 결과를 만든다.

사용 예시

API로 사용할 때는 모델 ID gemini-3.1-flash-image를 지정하면 된다. 한 번의 프롬프트에 최대 14장의 참고 이미지를 입력할 수 있고, 출력은 512px·1K·2K·4K(프리뷰) 해상도와 16:9·9:16·1:1 등 14가지 종횡비를 지원한다. Google AI Studio에서 바로 시험해볼 수 있고, Gemini 앱·Google 검색·Google Ads에도 통합되어 있다.

이미지 편집 워크플로우에서는 기존 사진을 참조 이미지로 넣고 자연어로 지시하면 피사체를 보존한 채 배경 교체, 오브젝트 추가, 스타일 변환이 가능하다. 텍스트가 들어간 인포그래픽 생성이나 손 메모를 다이어그램으로 변환하는 작업에도 강하다.

모델 라인업 비교

Google의 이미지 생성 모델은 세 단계로 나뉜다. 초기 Nano Banana가 바이럴 화제를 낳았다면, Nano Banana Pro는 스튜디오급 품질을 목표로 한 상위 모델이다. Nano Banana 2는 그 사이에서 Pro 수준의 인텔리전스를 Flash 속도로 제공한다는 포지셔닝이다. Artificial Analysis Image Arena 텍스트-이미지 생성 부문에서 1위를 기록했다.

주의할 점

4K 해상도 출력은 2026년 기준 프리뷰 상태다. 출력 이미지는 토큰 기반으로 과금되며, 512px 기준 이미지 1장에 747 출력 토큰이 소비된다. 입력 이미지는 파일 1개당 인라인 업로드 시 최대 7MB, Google Cloud Storage 경유 시 최대 30MB까지 허용된다.
