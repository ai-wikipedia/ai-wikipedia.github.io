Aion-3.0 (AionLabs)

AionLabs가 만든 롤플레이·스토리텔링 특화 언어 모델. GLM 패밀리를 기반으로 여러 전문 모델이 협업해 한 응답을 생성하는 다중 모델 시스템이다.

어떤 모델인가

Aion 3.0은 단일 모델이 아니다. 여러 전문 모델이 하나의 응답을 함께 만들어 내는 협업 생성(collaborative generation) 구조를 채택했다. 각 모델이 서사 구조, 갈등, 긴장감 등 서로 다른 요소를 담당하고, 그 결과를 합쳐 최종 출력을 만든다. GLM 패밀리를 기반 모델로 사용하며, 크리에이티브 라이팅과 롤플레이에서 단일 모델 대비 더 풍부한 내러티브를 만들어 내는 데 초점을 맞췄다.

컨텍스트 윈도우는 131,072 토큰, 최대 출력은 33,000 토큰이다. 가격은 입력 $3/1M 토큰, 출력 $6/1M 토큰이며, OpenRouter를 통해 API로 접근할 수 있다.

사용 예시

OpenAI 호환 API를 제공하기 때문에, 기존 GPT 기반 앱에서 엔드포인트와 API 키만 바꿔서 바로 붙일 수 있다. 인터랙티브 픽션 게임, TRPG 보조 도구, 장편 소설 협업 도구처럼 서사의 일관성과 긴장감이 중요한 프로젝트에 적합하다. 성인 테마나 복잡한 도덕적 갈등이 있는 시나리오처럼 일반 모델이 회피하는 영역도 다룰 수 있도록 설계되어 있다.

주의할 점

협업 생성 구조 특성상 레이턴시가 단일 모델 대비 높을 수 있다. 창작 외 일반 작업(코딩, 분석 등)에서는 범용 모델 대비 이점이 크지 않다. 경량 버전인 Aion 3.0 Mini도 같은 구조로 제공된다.
