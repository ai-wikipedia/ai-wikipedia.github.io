클로드 소네트 5

Claude Sonnet 5 (Anthropic)

Anthropic의 Sonnet 시리즈 최신 모델로, Sonnet 가격에 Opus에 가까운 성능을 제공하도록 설계된 에이전트 특화 모델이다.

개념 설명

Claude Sonnet 5는 2026년 6월 30일 Anthropic이 출시한 Sonnet 시리즈의 최신 모델이다. 핵심 컨셉은 하나다 — Opus급 성능을 Sonnet 가격에 제공하는 것. 출시 당일 Free·Pro 계정의 기본 모델로 바로 전환됐을 만큼 Anthropic이 자신하는 미드티어 모델이다.

이전 Sonnet 모델과 가장 큰 차이는 에이전틱 능력이다. Anthropic은 Sonnet 5를 "가장 에이전틱한 Sonnet"으로 소개했다. 브라우저를 열고, 터미널을 실행하고, 계획을 세워 자율적으로 긴 작업을 처리하는 능력이 몇 달 전까지만 해도 Opus급 모델에서나 가능했던 일이다.

사용 예시

Claude Code에서 서브에이전트로 쓸 때 Sonnet 5가 빛난다. 에이전틱 코딩 벤치마크에서 63.2%를 기록해 Sonnet 4.6(58.1%)보다 향상됐고, Opus 4.8(69.2%)과의 격차도 6%포인트로 좁혔다. 지식 작업에서는 Opus 4.8을 소폭 앞서기도 한다. API 모델 ID는 claude-sonnet-5이며, AWS Bedrock에서는 anthropic.claude-sonnet-5, Vertex AI에서는 claude-sonnet-5로 접근한다. claude.ai, Claude Code, AWS Bedrock, Microsoft Foundry에서 모두 사용 가능하며, 가격은 입력 100만 토큰당 $3, 출력 $15다(2026년 9월 이후 기준).

안전 설계

Sonnet 5는 출시와 함께 사이버보안 능력을 의도적으로 제한했다. Mythos급 모델이 Firefox 147 취약점 익스플로잇 벤치마크에서 높은 점수를 기록하는 반면, Sonnet 5는 이 수치를 훨씬 낮게 유지했다 — 정부 모델 검토를 통과하고 출시될 수 있었던 이유 중 하나다. 사이버 가드레일이 기본 활성화된 상태로 배포됐다.

주의할 점

커뮤니티 반응은 엇갈린다. 일부 사용자는 Sonnet 4.6 대비 속도 저하와 세션 사용량 증가를 보고했다. AutomationBench에서는 5.3%에서 13.5%로 큰 폭의 향상이 있었지만, 실제 워크플로우에서의 체감은 다를 수 있다. 복잡한 코딩 작업에는 여전히 Opus 4.8이 더 나은 선택일 수 있다.
